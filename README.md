# The Witch Project

Proyek game platformer sederhana yang dikembangkan menggunakan Godot Engine.

## Informasi Versi
- **Godot Version:** 4.6.1.stable

## Fitur Utama
- Sistem pergerakan player (jalan, loncat, double jump).
- Sistem kunci dan peti:
  - Player hanya bisa membawa 1 kunci dalam satu waktu.
  - Kunci dapat digunakan untuk membuka peti mana pun.
  - Animasi peti berhenti pada frame tertentu saat terbuka.
  - Kunci akan menghilang (pindah koordinat) setelah digunakan.
